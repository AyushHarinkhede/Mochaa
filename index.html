String Methods

Number string methods in JavaScript are used to convert numbers to strings or manipulate string representations of numbers. Some common methods include:

  • toString(): Converts a number to a string.
  • toFixed(digits): Formats a number to a fixed number of decimal places.
  • toPrecision(precision): Formats a number to a specified precision.

Example: (123.456).toFixed(2) // "123.46"

Number String Methods

String methods in JavaScript allow you to work with strings, such as searching, replacing, and extracting parts. Some common methods include:

  • length: Returns the length of the string.
  • charAt(index): Returns the character at a specified index.
  • substring(start, end): Extracts a part of the string.
  • replace(search, replace): Replaces text in the string.
  • toUpperCase(): Converts to uppercase.
  • toLowerCase(): Converts to lowercase.

Example: "hello".toUpperCase() // "HELLO"

Array Methods

Array methods in JavaScript are used to manipulate arrays, such as adding, removing, or transforming elements. Some common methods include:

  • push(item): Adds an item to the end of the array.
  • pop(): Removes the last item from the array.
  • shift(): Removes the first item from the array.
  • unshift(item): Adds an item to the beginning of the array.
  • slice(start, end): Extracts a portion of the array.
  • splice(start, deleteCount, ...items): Changes the contents of an array.
  • forEach(callback): Executes a function for each array element.
  • map(callback): Creates a new array with results of calling a function for every element.
  • filter(callback): Creates a new array with elements that pass a test.

Example: [1,2,3].map(x => x*2) // [2,4,6]
